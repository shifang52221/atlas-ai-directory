# AI Agents Vertical Site Design (English)

**Date:** 2026-03-09  
**Scope:** Build an English vertical AI navigation/decision site with affiliate-first monetization and AdSense as secondary.

## 1. Positioning

- **Category:** AI Agents and Automation tools for SMB operators.
- **Primary value:** Help users choose and implement the right tool stack fast.
- **Differentiator:** Scenario-first decision content, not generic tool listing.

## 2. Target Users

- Small business operators and solo teams.
- Non-technical growth/operations users who need automation quickly.
- Buyers comparing cost, setup effort, and integration fit.

## 3. Core Jobs To Be Done

- "I need to automate a specific ops task this week."
- "I need the best tool under my budget and skill level."
- "I need a clear comparison and implementation path."

## 4. Information Architecture

- `/` Home: value proposition + top use cases + trust signals.
- `/use-cases` and `/use-cases/[slug]`: task-driven entry pages.
- `/compare` and `/compare/[slug]`: A vs B vs C comparisons.
- `/tools` and `/tools/[slug]`: tool profile with pricing and alternatives.
- `/workflows` and `/workflows/[slug]`: practical automation templates.
- `/updates`: price/model/news changes relevant to decisions.
- `/about`, `/contact`, `/disclosure`, `/privacy-policy`, `/terms`.

## 5. Content Model

Each tool entry includes:

- Summary, ideal user, setup time.
- Pricing snapshot and billing model.
- Integrations, API/automation support.
- Pros/cons, alternatives.
- Affiliate status and disclosure label.
- Last tested date and evidence notes.

Each comparison page includes:

- Decision table with weighted criteria.
- "Best for X" recommendations.
- Short implementation checklist.

## 6. Monetization Strategy

- **Primary (Phase 1):** Affiliate links in high-intent pages (`best`, `vs`, `review`, `alternatives`).
- **Secondary (Phase 2):** AdSense after stable quality traffic and policy readiness.
- **Later (Phase 3):** Featured placements and sponsored deep reviews.

## 7. SEO and GEO Strategy

- Focus on high-intent query clusters:
  - "best ai agent for [job]"
  - "[tool a] vs [tool b]"
  - "[tool] alternatives"
- Use structured data:
  - `SoftwareApplication`
  - `Review`
  - `ItemList`
- Enforce editorial freshness:
  - visible last-updated date
  - monthly review pass on top pages

## 8. Trust and Compliance

- FTC-style affiliate disclosure on monetized pages.
- Policy pages from day one: privacy, terms, disclosure.
- Clear separation of editorial criteria vs commercial placements.

## 9. MVP Definition (First 4 Weeks)

- Launch core site with:
  - 1 homepage
  - 8 use-case pages
  - 12 comparison pages
  - 20 tool pages
  - 5 workflow pages
  - required policy pages
- Implement analytics event tracking for affiliate clicks.
- Build a repeatable content template system.

## 10. Success Metrics (First 90 Days)

- Affiliate click-through rate from money pages.
- Number of pages indexed and ranking in top 20.
- Monthly affiliate conversions and revenue.
- Returning users from workflow and comparison pages.
